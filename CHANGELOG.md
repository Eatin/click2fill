# Changelog

## v1.0.3 2025-12-30

* 新增子文档插入功能，支持将API响应插入到新创建的配套知识子文档中
* 修复子文档内容重复渲染问题，确保内容只显示一次
* 支持子文档模板渲染，允许在子文档中使用${message}等模板变量
* 修复子文档引用关联失败问题，移除严格的文档ID验证，提高异步操作的可靠性
* 重构了代码结构，提取了共享的getValueByPath辅助方法
* 创建了专门的renderTemplate方法用于子文档渲染
* 优化了handleMenuClick方法，根据插入方式选择不同的渲染逻辑

## v1.0.2 2025-12-23

* 改进了菜单添加页面，提供更清晰的UI布局和更好的用户体验
* 为关键词和正则表达式字段添加了说明文字，解释其用途和使用方法
* 明确了关键词和正则表达式为可选字段
* 将headers和params配置移到了高级设置折叠面板中
* 将"插入模板"改为"API请求结果渲染"
* 为API请求结果渲染添加默认值"${data}"，并支持使用${data}表示整个API返回数据
* 更新了空菜单提示文本，改为更明确的"无菜单配置，请先点击‘新增请求’"
* 简化了请求方法选项，只保留GET和POST
* 设置POST为默认请求方法
* 为请求参数添加了包含${selectText}的占位符
* 优化了配置页面的按钮文本，改为"导入"、"导出"、"新增请求"
* 修复了${selectText}占位符的处理逻辑，确保它能被实际选中的文本替换

## v1.0.1 2025-12-22

* 版本号从v0.4.3升级到v1.0.1，标记为正式发布版本
* 解决了配置过多显示的问题，通过预先根据选中的文本来展示菜单
* 优化代码结构，提高插件性能
* 修复已知问题，增强插件稳定性

## v0.4.3 2025-12-18

* 移除插件设置面板中的重复配置按钮，只保留悬浮菜单中的配置选项
* 修改内容插入逻辑，将返回内容追加到选中文本后面而不是替换
* 优化URL处理，自动添加协议前缀并记录详细请求日志便于调试

## v0.4.2 2025-12-17

* 修复动态菜单显示问题，使用正确的Menu API
* 调整菜单位置为光标位置而非鼠标位置
* 添加b3-menu__icon CSS类优化图标显示大小

## v0.4.1 2025-12-16

* 移除工具栏图标集成
* 修改快捷键为⇧⌘I
* 保留快捷键功能，通过快捷键打开动态菜单

## v0.4.0 2025-12-15

* 实现Click2Fill核心功能
* 支持动态配置请求URL、菜单自定义
* 添加数据填充到文档功能
* 实现国际化支持
