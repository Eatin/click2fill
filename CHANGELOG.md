# Changelog

## v1.0.1 2025-12-22

* 版本号从v0.4.3升级到v1.0.1，标记为正式发布版本
* 解决了配置过多显示的问题，通过预先根据选中的文本来展示菜单
* 优化代码结构，提高插件性能
* 修复已知问题，增强插件稳定性

## v0.4.3 2025-12-18

* 移除插件设置面板中的重复配置按钮，只保留悬浮菜单中的配置选项
* 修改内容插入逻辑，将返回内容追加到选中文本后面而不是替换
* 优化URL处理，自动添加协议前缀并记录详细请求日志便于调试

## v0.4.2 2025-12-17

* 修复动态菜单显示问题，使用正确的Menu API
* 调整菜单位置为光标位置而非鼠标位置
* 添加b3-menu__icon CSS类优化图标显示大小

## v0.4.1 2025-12-16

* 移除工具栏图标集成
* 修改快捷键为⇧⌘I
* 保留快捷键功能，通过快捷键打开动态菜单

## v0.4.0 2025-12-15

* 实现Click2Fill核心功能
* 支持动态配置请求URL、菜单自定义
* 添加数据填充到文档功能
* 实现国际化支持
