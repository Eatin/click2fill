## ✨ 亮点功能

1. **核心文本填充功能**：通过选择文本并使用快捷键 `⇧⌘I`，使用API响应自动填充文本
2. **动态菜单**：根据配置显示自定义菜单选项
3. **模板支持**：支持使用 `${key}` 语法的简单模板渲染
4. **配置管理**：提供直观的配置面板管理API菜单
5. **自动清理**：卸载时自动清理配置文件

## 🚀 使用方法

1. 选择要处理的文本
2. 使用快捷键 `⇧⌘I` 打开动态菜单
3. 选择配置好的API菜单
4. 插件将调用API并自动填充结果

## 📦 安装

通过 SiYuan 插件市场搜索 "Click2Fill" 或 "文本填充助手" 进行安装，或手动下载最新版本的 ZIP 包导入。

## 📝 注意事项

- 确保配置的API URL包含完整的协议前缀（http:// 或 https://）
- 插件默认使用 JSON 格式的响应数据
- 可以通过配置模板自定义输出格式
- 卸载插件时会自动清理所有配置数据
